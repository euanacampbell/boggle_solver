# Boggle Solver

A recursive method for finding all words in a given square grid.

## Installation

No non-standard libraries needed.

## Usage
Grid can be updated within the main.py file.

```bash
>> python3 main.py
['M', 'A', 'P']
['E', 'T', 'E']
['D', 'E', 'N']

completion:
1/9
...
9/9
complete

MET
MAT
...
PATEN
TEAMED

Words found: 46
Shortest word: MET
Longest word: TEAMED
```
